LINE 群組通知設定

如何設定中繼 Webhook 發送至 LINE？
  1. 建立一個 LINE Bot 帳號並將其加入您的 LINE 工作群組。
  2. 建立一個免費的 Google Apps Script (GAS) 或使用 Make.com / IFTTT 接收此網頁的 POST 請求。
  3. 中繼腳本呼叫 LINE Messaging API 的 push 介面，將訊息轉傳至群組。
  4. 早上 10:00 定時日報：系統會在管理員登入且時間超過 10:00 時，若當日未發送則自動觸發；若需無須網頁開啟的完全自動化，可在 GAS 中設定時間驅動程序（Time-driven trigger）於每日 10:00 執行。
  5. 若未啟用或未填寫 Webhook 網址，系統亦會於右上角自動彈出 LINE 群組訊息模擬畫面。
